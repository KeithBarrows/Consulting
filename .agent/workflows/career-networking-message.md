description: Draft a concise LinkedIn connection message to a hiring manager
---

This workflow helps draft a networking message to connect with hiring managers or professionals.

1. Ask the USER for the target [Company], [Role], a recent [Topic] or project of interest, and the USER's relevance [Skill].
2. Execute the following instruction:
"Draft a concise (under 300 characters) LinkedIn connection message to a hiring manager at [Company] for a [Role] position. Mention my interest in their recent project on [Topic] and connect it to my experience in [Skill]."
3. Present the drafted message to the USER.
4. Ask the USER, "What is missing to make this 10/10?" and iterate on the feedback.
