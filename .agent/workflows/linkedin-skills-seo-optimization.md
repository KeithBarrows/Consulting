---
description: Extract top 10 hard and soft skills for a target role to add to a LinkedIn profile
---

This workflow identifies missing skills based on a target job description to improve LinkedIn searchability.

1. Review the USER's existing skills directly from `C:\Repos\Consulting\LinkedIn\Skills.csv` to know what they already have on their profile.
2. Ask the USER for the [Target Title], the [Company Type] (e.g., tech startup, large enterprise), and the target job description.
2. Execute the following instruction:
"Based on the job description for a [Target Title] at [Company Type], list the top 10 hard and soft skills I should add to my LinkedIn profile to maximize my searchability for that role".

3. Present the list of 10 hard and soft skills to the USER.
4. Advise the USER to append those skills onto their LinkedIn profile's top skills section.
5. Ask the USER, "What is missing to make this 10/10?" and iterate on the feedback.
